SHORT DESCRIPTION OF THE APPLICATION

The login system includes functions such as:

- logging into the application

- registration,

- changing the password using the mailbox,

Application using relationship OneToOne and ManyToOne in database.

A application was designed to allow the definition of an employee using SessionWizardView and editing, showing and deleting data. Application also collecting arhivate data and show history change.

Structure of data designed based on polish standard. For example PESEL (is polish identifier number includes 11 digits), post code in format 11-111.

The sample validation for the pesel number and for the postal code was used in the project. PESEL number is validated based on date of birth (contains only didgits and has 11 characters). The system allows the employee to be entered without a PESEL number.

List of profession is not translated to english.
